```bash
  xpen fetch http://codepen.io/pimskie/pen/lvwhs
  xpen f http://codepen.io/pimskie/pen/lvwhs
```

抓取一个线上 codepen 项目到本地。

```bash
  xpen init
  xpen i
```

在本地创建一个 xpen 项目。

```bash
  xpen serve
  xpen s
  xpen s -p 2014
```

新建服务器编辑 js， css， html 文件

```bash
  xpen upload codepen
  xpen u c
```
